# Game Portal (网页游戏大厅) 🎮

这是一个充满复古乐趣与现代创新的网页游戏集合大厅！本项目包含了 5 款由 HTML5 Canvas 和原生 JavaScript 驱动的经典游戏，支持中英文双语界面（Bilingual UI）。无需安装，点开即玩！

## 🌟 游戏列表 (Games List)

### 1. 末日枪手 (Mech Shooter) 🔥 *强烈推荐*
一款支持双人合作（Co-op）的满屏割草射击游戏！
- **真·全屏体验**：游戏画面自适应浏览器大小，无缝割草。
- **智能双人模式**：P1 (`WASD`移动 + `Space`射击) | P2 (`方向键`移动 + `Enter/小键盘Enter`射击)。
- **武器升级系统**：通过击杀机械怪解锁双枪、散弹枪、步枪和极光射线。
- **机械狂潮与巨型变异 Boss**：面对各种履带式科幻机器人的围剿，游戏引入了终极挑战——10 只形态各异的“巨型生化机械蜘蛛”Boss 会在地图边缘生成并逐渐向你逼近。
- **全景雷达系统**：界面拥有 10 个独立导航箭头，实时追踪并标记不同颜色的终极 Boss 位置及其距离。
- **盲盒空投 (Loot Boxes)**：地图上随机刷新宝箱，提供急救包、无敌护盾、全屏秒杀核弹，甚至是扣血陷阱！
- **AI 智能托管机制**：玩家如果超过 5 秒钟不操作，聪明的 AI 将会自动接管角色，进行风筝走位并全自动射击！玩家按任意键即可随时收回控制权。

### 2. 坦克大战 (Tank Battle)
重温红白机时代的经典回忆，保卫基地，消灭敌方坦克大军。

### 3. 贪吃蛇 (Snake)
最经典的休闲游戏，吃掉苹果让蛇变长，看看你能坚持多久不撞墙。

### 4. 三国无双射击 (Kingdom Rush)
一款支持 1-3 人同屏合作的古风割草射击游戏！
- **三国国风主题**：化身古代武将，抵御黄巾军、西凉铁骑和铁浮屠的进攻。
- **深度的武器进化系统**：在局内通过花费军饷升级武器，形态将发生质的飞跃：
  - 🏹 **1阶：长弓** (基础弓箭)
  - 🏹 **2阶：诸葛连弩** (攻速飞快的连发弩箭)
  - 🪨 **3阶：霹雳车** (发射巨大的落石，造成大范围真实AOE溅射)
  - 🪵 **4阶：八牛床弩** (发射巨型弩箭，无限穿透直线上所有敌人)
  - 🐉 **5阶：火龙出水** (发射自带智能追踪+范围爆炸的古代追踪火器)
  - 💣 **6阶：万人敌(震天雷)** (终极武器，威力绝伦的全屏核爆级清场)
- **三国名将 Boss 战**：极度强悍的名将（无双吕布、天公张角、魔王董卓）降临，且拥有智能的“左右横跳”AI避开玩家火力。
- **动态难度与经济系统**：支持1-10条独立兵线，路线越多刷怪越疯狂。敌人的血量无限成长，掉落的军饷也同步暴增！
- **丰富的奇兵道具（包含Buff与Debuff）**：击杀小兵随机掉落各种功能奇葩的道具，例如：草船借箭（全屏狂暴射击）、奇门遁甲（全员罚站）、鸩酒（扣除军饷）等。

### 5. 葫芦娃大战妖精 (Calabash Brothers)
经典的国漫 IP 塔防！种植各种葫芦娃来抵御蛇精和蝎子精的进攻。

## 🚀 如何运行 (How to Run)

本项目无需任何后端环境或构建工具（No Node.js/Webpack required）。
1. 直接在浏览器中双击打开 `index.html` 即可进入游戏大厅。
2. 也可以使用任何本地服务器运行（例如：`python -m http.server` 或 VSCode 的 Live Server 插件）。

## 💡 操作指南 (Controls)

每款游戏内都提供了详细的按键说明。大多数游戏支持使用键盘的 `W A S D` 或 `↑ ↓ ← →` 进行移动。

## 🌍 双语支持 (Bilingual)
游戏大厅以及所有的游戏界面均已实现中英双语显示，方便不同国家和地区的玩家一起享受游戏的乐趣。
