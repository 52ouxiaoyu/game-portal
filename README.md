# 🎮 游戏中心

个人网页游戏合集，收录多款原创 HTML5 小游戏，即点即玩，无需下载。

## 🌐 在线访问

**https://game-portal-4iy.pages.dev/**

## 🎯 游戏列表

| 游戏 | 类型 | 说明 |
|------|------|------|
| 🔫 [坦克大战](./坦克大战/) | 射击 | 经典坦克大战，支持双人对战，1000关随机生成 |
| 🐍 [贪吃蛇](./贪吃蛇/) | 休闲 | 经典贪吃蛇，多种玩法模式 |
| 🏰 [王国保卫战](./王国保卫战/) | 塔防 | 策略塔防游戏，建造防御塔抵御敌人 |
| 🏮 [葫芦兄弟](./葫芦兄弟/) | 冒险 | 致敬经典动画，七兄弟各有神通 |

## 🚀 技术栈

- 纯 HTML / CSS / JavaScript
- 零依赖，无需构建工具
- Canvas 渲染，流畅运行
- 响应式设计，支持手机/平板/电脑

## 📱 支持平台

- ✅ PC 浏览器（Chrome / Firefox / Safari / Edge）
- ✅ 手机浏览器
- ✅ 平板设备

## 🎨 特色功能

- 🤖 AI 自动托管（坦克大战）
- 👥 双人对战模式
- 🎵 音效系统
- 💾 最高分记录
- 📱 触屏操控支持

## 🛠️ 本地运行

```bash
# 克隆仓库
git clone https://github.com/52ouxiaoyu/game-portal.git

# 进入目录
cd game-portal

# 启动本地服务器
python3 -m http.server 8080

# 打开浏览器访问
# http://localhost:8080
```
