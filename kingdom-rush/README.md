# 🎮 王国保卫战 - Kingdom Rush

这是一款全新的双人合作射击防守游戏！

## 🎯 游戏玩法
- **多人合作**：支持 1 到 3 名玩家共同防守！
- **动态路线**：敌人可以从 1 到 5 条路线发起进攻，你可以在开始菜单自由配置难度！
- **自动射击**：将军们会自动向正上方射出箭矢。
- **技能道具**：击杀敌人会掉落各种神奇道具（炸弹、冰冻、宝箱），射中道具即可全屏释放技能。
- **共享军费升级**：共同收集金币，一键升级全军攻击力、射速与多重箭！
- **最后防线**：任何敌人触碰底部的城堡，游戏即刻失败！

## 🕹️ 操作说明
在主菜单调整好**玩家人数**和**敌人路线**后点击开始：
- **玩家 1 (金色将军)**：`A` / `D` 键
- **玩家 2 (青色将军)**：`⬅️` / `➡️` 方向键
- **玩家 3 (紫色将军)**：`J` / `L` 键

## 🚀 立即体验
直接使用浏览器打开 `index.html` 开始游戏！
