# 🎮 王国保卫战 - Kingdom Rush

这是一款全新的双人合作射击防守游戏！

## 🎯 游戏玩法
- **双将并肩**：玩家1与玩家2分别控制两名将军，在屏幕下方抵御不断落下的敌人。
- **自动射击**：将军们会自动向正上方射出箭矢。
- **技能道具**：击杀敌人会掉落各种神奇道具（炸弹、冰冻、宝箱），射中道具即可全屏释放技能。
- **共享军费升级**：共同收集金币，一键升级全军攻击力、射速与多重箭！
- **最后防线**：任何敌人触碰底部的城堡，游戏即刻失败！

## 🕹️ 操作说明
- **玩家 1 (金色将军)**：
  - `A` / `D` 键：左右移动
- **玩家 2 (青色将军)**：
  - `⬅️` / `➡️` 键盘方向键：左右移动

## 🚀 立即体验
直接使用浏览器打开 `index.html` 开始游戏！
