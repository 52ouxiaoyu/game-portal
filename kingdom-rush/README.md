# 🎮 像素塔防 - Pixel Tower Defense

**[点击这里直接玩游戏](https://52ouxiaoyu.github.io/Kingdom-Rush/)**

一款原创像素风格塔防游戏，灵感来源于经典塔防类型。

## 游戏特色

- **4种防御塔**：弓箭塔、法师塔、炮塔、冰塔
- **5种敌人**：哥布林、兽人、巨魔、暗黑骑士、飞龙
- **2个英雄**：战士（旋风斩）、法师（暴风雪）
- **3个关卡**：翡翠森林、荒芜之地、暗影城堡

## 操作方式

- 点击右侧面板选择塔 → 点击地图放置
- **空格** 开始下一波
- **1-4** 快速选择塔
- **ESC** 取消放置
- **P** 暂停

## 运行方式

直接打开 `index.html` 或通过本地服务器访问：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`

## 技术栈

- HTML5 Canvas
- 原生 JavaScript
- Web Audio API（音效）
